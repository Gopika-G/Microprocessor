# Macroprograms
assemble the programs using following commands
## Linux Assembler
commands in terminal
```
nasm -felf64 hello.asm
```
load it with
```
ld hello.o
```
and execute it with our own 
```
a.out
```
## Windows Assembler
commands in cmd for 32bit systems or dos
```
masm hello.asm
```
load it with
```
link hello.obj
```
and execute it with our own 
```
hello.exe
```
